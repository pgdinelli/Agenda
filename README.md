# Agenda
Projeto Agenda consiste em uma aplicação web onde o usuário pode cadastrar novos contatos em uma agenda, editar dados de contatos cadastrados ou excluir contatos salvos. O sistema também possui criação de conta e login do usuário para poder executar as ações de cadastro de contatos. Este projeto foi desenvolvido com JavaScript, Node.js, EJS e Bootstrap, utilizando express para rodar o servidor e mongoose para conexão com banco de dados MongoDB.
